# Online and Computer Safety
Online and Computer Safety class. High level, 10 hours for middle and high school level students

Each day will have some example security issues and/or scams sprinkled over a topic area. Since the systems used change so quickly, there will not be specific instructions. Instead, we will go over things to think about and quick examples of the type of thing covered.
